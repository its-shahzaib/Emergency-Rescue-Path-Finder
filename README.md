# Emergency Rescue Path Finder

## Project Description
Emergency Rescue Path Finder is a Python mini project developed for an AI/ML internship. The purpose of this project is to help a rescue team find the best route from a rescue station to an emergency location.

The city map is represented using a dictionary graph where locations are nodes and roads have distance costs.

## Concepts Used
- Python basics
- If-else statements
- Loops
- Lists
- Dictionaries
- Functions
- BFS
- DFS
- UCS
- A* Search Algorithm

## Features
- Takes start and emergency location from the user
- Validates user input
- Finds route using BFS, DFS, UCS, and A*
- Compares algorithms based on path, cost, and explored nodes
- Displays the best route for emergency rescue

## Algorithms Used
1. BFS - Breadth First Search
2. DFS - Depth First Search
3. UCS - Uniform Cost Search
4. A* Search Algorithm
